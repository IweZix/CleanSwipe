# SwipeFiles

SwipeFiles is a desktop application inspired by Tinder for managing your files. Swipe left to delete or right to keep, and quickly organize your downloads or folders. Built with **Tauri**, **React**, and **Chakra UI**, it offers speed, simplicity, and a modern interface.
